# 🎮 Game X Innovate – HACKSPRINT v7.0

> 🧠 Psychological Mystery Adventure Game built in 24-hour Hackathon

---

## 🚀 Overview
**Game X Innovate** is a **single-player psychological mystery adventure** where the player is trapped in a surreal mirror world and must escape by solving puzzles, navigating obstacles, and uncovering hidden truths.

---

## 🧩 Theme
**Open Innovation**

---

## 🎯 Objective
Escape the **Mirror World** using a secret passkey by:
- Solving puzzles 🧠  
- Navigating traps ⚠️  
- Fighting enemies 👾
